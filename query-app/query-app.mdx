Welcome to Dune, where we have created the perfect experience for querying blockchain data. Our platform consists of two main parts, the **Dune Web App** and the **Query Engine**, each designed to cater to your data needs in a unique and powerful way.

The **Dune Web App** is your interface to the full power of dune's capabilities. Search for tables, create queries and visualize the results in a user-friendly environment. 
The **Query Engine** is the core of Dune, where you can write complex SQL queries to extract and analyze blockchain data.

## Dune Web App

The Dune Web App is an intuitive, user-friendly interface designed to bring the expansive world of blockchain data to your fingertips. It is the visual layer that allows you to interact with the data in an accessible way, enabling you to:

- **Create Queries**: Use the Query Editor to craft SQL queries that extract the data you need for analysis.
- **Visualize Data**: Create and customize charts and dashboards with various graphical tools to make data easy to understand and actionable.
- **Collaborate**: Work together with team members, share your findings, and access shared insights from the Dune community.
- **Create and Share Reports**: Generate reports and visualizations that can be shared with stakeholders to inform decision-making.

## Query Engine

Delve deeper into data analysis with the Dune Query Engine. This is the core of where data processing and complex analysis happen. It allows for sophisticated data manipulation and extraction, using SQL to craft queries that can process vast amounts of data for in-depth insights. Here, you'll find:

- **Advanced SQL Features**: Utilize advanced SQL functions and operators to perform complex data analysis tasks.
- **Optimization Techniques**: Learn how to write efficient queries that minimize processing time and maximize performance.
- **Data Management**: Handle large datasets with ease, using features like materialized views and query scheduling to manage your data workflow effectively.
